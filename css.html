What is CSS?

CSS stands for Cascading Style Sheets.
It is a style sheet language which is used to describe the look and formatting of a document written in markup language.
It provides an additional feature to HTML. It is generally used with HTML to change the style of web pages and user interfaces.

How to insert CSS

CSS can be inserted in an html code in the following ways

  1. Inline CSS; this where the style attribute is used, it inserted inside the html
  2. CSS; this is where the style element is used, especially in the head section, and the desired place is targeted
  3. External CSS; this is mostly used type,
    in this a separate file is created and target elements, classes and id, it can be used for more than one page
